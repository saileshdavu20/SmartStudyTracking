# 📚 Smart Study Tracker (Java Project)

## 🚀 Project Overview

Smart Study Tracker is a simple console-based Java application that helps users track their daily study activities.
Users can add subjects, record study hours, view all entries, calculate total study time, and search records.

---

## ✨ Features

* ➕ Add Study Entry (Subject & Hours)
* 📋 View All Study Records
* ⏱️ Calculate Total Study Hours
* 🔍 Search by Subject
* ❌ Delete Study Entry
* ⚠️ Input Validation (handles invalid inputs)

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts (Classes & Objects)
* ArrayList (Dynamic Storage)
* Scanner (User Input)

## ▶️ How to Run

1. Open project in Eclipse / IntelliJ
2. Run `Main.java`
3. Follow menu options:

```
1. Add Study
2. View All
3. Total Hours
4. Search by Subject
5. Exit
```
